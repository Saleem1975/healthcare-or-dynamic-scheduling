# Figure Guide

Recommended figure placement in the GitHub README:

1. `project11_waiting_time_comparison.jpg`
   - Place near the top of the README under **Business Impact**.
   - Shows patient waiting time reduction from traditional static scheduling to CNN-based dynamic scheduling.

2. `05_data_pipeline_framework.jpg`
   - Place in the **Data Pipeline** or **Methodology** section.
   - Explains how laparoscopic videos move through frame extraction, preprocessing, augmentation, CNN training, and testing.

3. `01_cnn_dynamic_scheduling_workflow.jpg`
   - Place in the **Proposed Solution** section.
   - Shows how real-time video frames are classified and used to update the operating room schedule.

4. `02_model_training_accuracy_summary.jpg`
   - Place in the **Model Training** section.
   - Summarizes training and validation accuracy behavior.

5. `03_confusion_matrix_redesigned.jpg`
   - Place in the **Model Evaluation** section.
   - Shows classification performance for end-of-Phase-4 detection.

6. `04_roc_curve_auc_095.jpg`
   - Place in the **Model Evaluation** section.
   - Highlights the reported ROC performance with AUC = 0.95.
